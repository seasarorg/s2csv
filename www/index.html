Welcome to the "s2csv" project.

S2CSVはSeasarとアノテーションを使ってCSVファイルやCSVデータを手軽に扱うためのライブラリです。

まだまだ業務で使用されるCSVファイルを手軽に扱えるように開発しました。

S2CSVファイルを使えば以下のようなことが出来ます。
・CSVファイル と Java Objectの相互変換を簡単に行える
・CSV個別の設定ファイルが必要ない
・バリデーションを行うことが出来る
・バリデーションメッセージ定義はStrutsライクでなじみがある形式
